## Local Development

Environment requirements:

- [Matlab](https://www.mathworks.com/products/matlab.html) version R2021a or later

### 1. Clone the repository

```shell
git clone https://github.com/cworld1/matlab-learning.git
cd matlab-learning
```

### 2. Run the code

```shell
matlab
```
